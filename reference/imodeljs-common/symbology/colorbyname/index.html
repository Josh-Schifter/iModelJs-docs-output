ColorByName Enumeration

A set of known colors by name, as a 32-bit integer in the form 0xBBGGRR (red is the low byte). This is different than color values in #RRGGBB format for HTML pages (red and blue are swapped).

note If your colors don't look right, likely you're using 0xRRGGBB where ColorDef expects 0xBBGGRR.
