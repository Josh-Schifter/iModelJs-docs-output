API Reference > geometry-core > CartesianGeometry > Ray3d

Ray3d Class

A Ray3d contains

  • an origin point.
  • a direction vector. The vector is NOT required to be normalized.
    • an optional weight (number).

Implements

  • BeJSONFunctions
